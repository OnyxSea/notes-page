HTML Forms

We want to update the Contact Details Page, so people can message us using the form

  • Go to mdn web docs
  • Search for the Form element
  • A form needs more than just HTML, because it needs behaviour
  • That requires Javascript

So how do we go about adding the form?

  • Add a brand new form using the (form class="" action="index.html" method="post")
  • We go inside the form tag and create a label
    • Where (label)Your Name:(/label)
    • This creates a Box after "Your Name:"
    • Afterwards specify input type
    • (input type="text" name="" value="")
  • Now we want a submit button
  • It's an input with a type="submit" instead of type="text"
    • Where (input type="submit" name="")
    • We can add multiple functionalities, including colour as an input type, this is where type="color"
    • You can also add checkboxes, type="checkbox"
    • Password inputs can be done with type="password", nobody can see your password when typing
  • Form has a closing tag at the end
  • Labels are texts the user sees
  • Inputs have no closing tags, but you can define them using the type attribute
  • When the input type="submit", it becomes a button
