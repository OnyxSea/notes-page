Forms in Practice - Create a Contact Me Form

Now we want to structure our form so it looks like a proper Contact Me Form.

  1. We want to remove password, replace with email
  2. New input type = email
  3. Also remove the input type="color"
  4. We want to ask them about their email, and the text Message
  5. We need to use something outside of input to make the Message larger
  6. It's a HTML text element named text area
  7. Follows the following syntax
    • (textarea name="name" rows="10" cols="30")
    • You can define the initial default sizes
    • If you click submit, it brings you back to the home page, because we linked the form there
  8. How can we have it publish into the internet?
    • Change the action into "mailto:info@londonappbrewery.com"
    • Wouldn't work, unless you have a mail application installed by default
  9. Go to Chrome Settings, set sites can ask to handle protocols
  10. Open a new tab, head to gmail.com
  11. Click allow mail.google.com to handle all links
  12. At the moment, all the text that emerges when you attempt to email is weird. To solve this, we need an encoding type
    • From the "form", remove class="", add enctype="text/plain"
    • This way, all the text will be plain text
  13. Then you can change the email accordingly before emailing
