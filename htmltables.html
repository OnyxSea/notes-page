HTML Tables Notes

  • We want a section to list our work experience
    • If we added our work experience with a paragraph tag
    • Adding it line by line makes it awkward
    • How do we make it nice with padding and sections?
    • we make a table
  • Tables can get really complicated
  • But it can be used to structure data in the website
  • Using tables, captions, colgroups, etc
  • This will make it look much nicer

Adding a table tag

  1. Add a (table) tag using table
  2. Use the (tr) tag to add the Table Row
  3. For each row, you want to add cells
  4. Use (td) tag to define the cell, then input the data
  5. Create a new table row and do the same for the next work experience entry
  6. Tables are structured such that they're indented into the same page

Table Heads, Table Bodies

  1. Use (thead) for table head
    • Define the table header cells here using (th)
    • Eg. Dates, Work
    • If you delete the (thead) tag it still works because of (th)
    • For each cell, use (td) to describe the cell contents
  2. Use (tbody) for table body
    • You want to place the body content inside (tbody), inside (td)
    • Tables can have outlines, eg cellpadding, cellspacing
    • But these are deprecated (no longer recommended to use)
    • This was used before they had CSS, and only had HTML
    • Use (table border="1") to add a border to the table
