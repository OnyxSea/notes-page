Publish Your Website!

Here we cover how we can publish the website for others to see. To do so we need to host it somewhere


So what do we do?

  • Many different hosting sites exist:
    • GoDaddy
    • BlueHost
  • But we don't want to pay
  • So how do we set it up?
  • We want to set it up with github
    • Essential to a developer's workflow
    • Login to github!

How do we use github??

  1. Create a new Repository
  2. Check the box which says "Add a README"
  3. Click "Create Repository"
  4. Repository is then created
  5. Upload all the corresponding files
  6. Give the version a name
  7. Commit the changes
  8. Go to settings, Github Pages
    • Enable it
    • Select main as your branch
    • It will be uploaded correspondingly
    • If there are any errors, it might be due to the filename, so change it to lowercase
