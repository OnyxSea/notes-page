    3. Create a row, inside there's 2 columns, with a skill eg iOS Development, and a second column, the emojis
    4. Copy and paste it, modify it for several entries
  • Challenge #3
    1. The solution is a nested list
    2. Create a borderless table
    3. Within the borderless table, there is a row and 2 columns
    4. Each column contains a table
      • In the table there are 2 rows
      • Each row has 2 columns, the skill, and the emoji rating
    5. Repeat for the 2nd column with a new table and do what was done above
